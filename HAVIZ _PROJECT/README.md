UCU CourierLite System.
The UCU CourierLite System is a simple delivery engine designed for UCU students to help them receive items from different hubs around and inside campus. It uses Python OOP concepts to load data, assign parcels to riders, and organize deliveries.

Hubs (Where items come from)
Haviz Cafe
JLuxe
Bobics
Ham Mukasa Library

Hostels (Delivery destinations)
Sabiiti, Nsibambi, David’s Ark, Skyz, Tupe, Ankrah, Luna, and others.

Riders
Sunday, Pascal, Timo, Matt, and Cole
Each rider delivers from their home hub and carries a limited load.

What the System Does
Loads hubs, parcels, riders from CSV files
Sorts parcels by priority (EXPRESS → NORMAL)
Assigns parcels to riders based on weight and home hub
Allows preview of how each rider’s parcels are arranged

Purpose
To make life easier for UCU students by improving delivery convenience between hubs and hostels while demonstrating key Python OOP concepts.
